# 🔐 Zero Trace

A full-stack web application with a **passcode-based access system**, deployed for public use.

🌐 **Live Demo:** https://zero-trace-bice.vercel.app/

---

## 🚀 About

Zero Trace is a secure web application that allows users to access features through a **passcode-based entry system**.

This project focuses on:
- Building a clean frontend experience  
- Deploying a backend service  
- Handling real-world integration between frontend and backend  

It demonstrates **end-to-end deployment**, making the app accessible across devices and networks.

---

## 🧠 Features

- 🔑 Passcode-based access system  
- 🌐 Fully deployed (Frontend + Backend)  
- ⚡ Fast and responsive UI  
- 🔗 Integrated frontend-backend communication  
- 📱 Accessible across devices
